# Referência Helicrash

Copie os arquivos da sua base para esta pasta:

```
C:\Users\gusta\OneDrive\Área de Trabalho\helicrash\
```

Para:

```
PC\reference\helicrash\
```

Arquivos importantes:
- fxmanifest.lua
- config.lua (ou shared/config.lua)
- server/*.lua
- client/*.lua
